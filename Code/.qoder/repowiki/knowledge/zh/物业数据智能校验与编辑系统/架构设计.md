- **核心架构**：采用“引擎-视图”分离模式。`src/engine` 封装纯逻辑校验器（简单/条件/跨行）与响应式状态存储（validationStore），通过 `window.luckysheet` API 与底层表格交互。
- **组件分层**：`TableEditor.vue` 作为容器集成 Luckysheet 实例并处理生命周期；`Toolbar`、`ValidationPanel` 等 UI 组件通过事件总线或 Store 同步状态。
- **校验流水线**：实现多级校验策略，包括输入即时格式检查（ON_INPUT）、失焦行级校验（ON_BLUR）以及延迟执行的跨行一致性校验（Cross-row）。
- **依赖管理**：内置 `dependencyTracker` 维护字段间的业务依赖关系，当源字段变更时自动触发目标字段的必填状态更新与重校验。
