# 条件检测逻辑（关键！）

## "有业主信息"的检测（OR逻辑）

```typescript
function hasOwnerInfo(row: number): boolean {
  const fields = [8, 9, 10] // 业主名称, 业主联系电话, 业主证件号码
  return fields.some(col => {
    const val = getCellText(row, col)
    return val !== ''
  })
  // ⚠️ 是OR逻辑！任一字段有值即返回true
  // 不是AND逻辑！不需要三个字段都有值
}
```

## "有租户信息"的检测（OR逻辑）

```typescript
function hasTenantInfo(row: number): boolean {
```
